# 知乎真实公式正文语料

本目录保存通过 `zhurl --web` 从公开知乎内容接口采集的公式语料索引。完整原始响应保存在本机忽略目录中，用于获授权环境下的离线重放，不随公开仓库分发。

- `raw/`：本机保存的 28 篇内容详情接口完整原始 JSON 响应，正文保留为原始 HTML；
- `manifest.json`：10 位答主及每篇内容的公开来源、哈希和公式数量；
- `formulas.json`：从原始正文派生的 3703 条去重 LaTeX 公式及来源。

语料固定纳入赵泠、酱紫君、卢健龙，并按公式数量补足其他公式密集的公开内容。候选发现过程、当前账号信息及登录凭据不属于公开语料。

`ZhihuFormulaCorpusTest` 始终校验公式索引并将全部去重公式交给应用当前依赖的 LaTeX parser；本机存在 `raw/` 时，还会校验原始响应哈希，重放完整 HTML 到 Markdown AST，检查 5598 次有效公式无丢失。

本机采集工具与完整原始响应一同保留但不公开分发。刷新属于显式操作，不能在普通测试中联网执行。
